# Epic 1: Rename three skill directories and clean up SKILL.md bodies

Covers SDD `## Requirements` SR-1 and SR-2 (including SR-2.5 for `bees-spec-review/SKILL.md`).

## Scope

**Note on line numbers below:** Line numbers are illustrative as of Epic authoring and may have drifted by execution time. Use `grep` to locate references before editing — the `grep` AC checks at the bottom of this Epic are the authoritative success signal.

1. Rename three skill directories (use `git mv` to preserve history):
   - `skills/bees-code-review/` → `skills/bees-engineer-review/`
   - `skills/bees-test-review/` → `skills/bees-test-writer-review/`
   - `skills/bees-doc-review/` → `skills/bees-doc-writer-review/`
2. For each renamed `SKILL.md`, update frontmatter and body:
   - Update `name:` (line 2) to match the new directory name.
   - Rewrite `description:` (line 3) to lead with the orchestrator scoping pattern: "Review the Engineer's diff during a `/bees-execute` or `/bees-fix-issue` review cycle. Returns a list of improvement work items for the orchestrator." Adapted per skill: "Engineer's diff" for `bees-engineer-review`; "Test Writer's test code" for `bees-test-writer-review`; "Doc Writer's documentation" for `bees-doc-writer-review`. The substrings "Standalone" and "ad-hoc" must not appear in the new descriptions.
   - Delete the `argument-hint:` line (line 4).
   - Delete the "When invoked standalone" body branch (line 13 in `bees-code-review` and `bees-test-review`; line 14 in `bees-doc-review`). Preserve the immediately-following "When invoked by `/bees-execute` or `/bees-fix-issue`" paragraph as the sole branch.
   - Update internal cross-references between the three SKILL.md files to use the new names: `bees-engineer-review/SKILL.md` line 39 (was `/bees-test-review`), `bees-test-writer-review/SKILL.md` line 32 (was `/bees-code-review`), `bees-doc-writer-review/SKILL.md` line 28 (was `/bees-code-review`).
3. Update `skills/bees-spec-review/SKILL.md` lines 9, 50, 51, 52 to reference the new names (per SR-2.5). One reference per line in the out-of-scope bullet block (lines 50–52); the conceptual-analogs paragraph at line 9 lists all three together.

## Acceptance criteria

- Three renamed directories exist; old directories do not.
- Each renamed `SKILL.md`'s `name:` matches its new directory name.
- `grep -r "Standalone\|ad-hoc\|invoked standalone" skills/bees-engineer-review/ skills/bees-test-writer-review/ skills/bees-doc-writer-review/` returns no matches.
- `grep -r "bees-code-review\|bees-test-review\|bees-doc-review" skills/bees-engineer-review/ skills/bees-test-writer-review/ skills/bees-doc-writer-review/ skills/bees-spec-review/` returns no matches.
