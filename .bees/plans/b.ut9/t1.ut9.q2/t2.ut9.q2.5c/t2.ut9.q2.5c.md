## Context

`/quo-test-writer-review` currently emits findings as a plain numbered list with **no severity tag** and **no fix-path/depth structure** (Step 4 "Generate Work Item List", `skills/quo-test-writer-review/SKILL.md` lines 150–178; line 154 explicitly states findings are *not* severity-tagged). This Task is the **first introduction of severity tagging** to this skill (SR-1.1) and adds the orthogonal per-fix-path depth dimension and fix-path enumeration on top. It is symmetric to the `/quo-engineer-review` (Epic `t1.ut9.9c`) and `/quo-doc-writer-review` (Epic `t1.ut9.jn`) extensions.

The two-dimensional finding emission (severity + per-fix-path depth + enumerated fix paths) is the foundational Phase-1 signal that the Phase-2 orchestrator routing table (sibling Epic `t1.ut9.no`) will consume. Depth tags become *emitted* here but are not yet *consumed* by any routing rule, so this Task ships standalone value with zero new user gates and zero `b.wii` exposure.

## What Needs to Change

- In `skills/quo-test-writer-review/SKILL.md` **Step 4 (Generate Work Item List)**, document the extended finding emission shape: every finding carries **exactly one severity tag** (`blocker` / `suggestion` / `nit`) AND **one or more enumerated fix paths**, each fix path carrying **its own depth tag** (`trivial-tweak` / `refactor-locally` / `re-architect`), following the SDD's "Finding shape with two-dimensional tagging" data model (authoritative source: Spec Bee SDD child `t1.tip.ux`, NOT the cumulative `docs/sdd.md`):
  - finding line: `` <n>. `<severity>` <one or more fix-path lines> — <description> ``

## Why

Severity describes *how important fixing-at-all is*; depth describes *what fixing costs*. The two dimensions are orthogonal, and the orchestrator's Phase-2 routing decision keys on depth (and the multi-path flag), not severity. Emitting this shape from `/quo-test-writer-review` is one of the four in-flow review skills' Phase-1 emission extensions; the routing table that consumes it is unimplementable without uniform emission across all four.
